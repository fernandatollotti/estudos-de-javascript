# Menu Dropdown

## 🚀 Descrição do projeto
Criei um menu responsivo com dropdown, projetado para proporcionar uma navegação intuitiva em diferentes dispositivos. Com um design moderno e interações suaves, o projeto facilita o acesso a diferentes seções e subcategorias.

## 📚 O projeto é composto pelas seções:
🔸 Menu principal com categorias e subcategorias dropdown
🔸 Barra lateral interativa para navegação em dispositivos móveis
🔸 Estilo responsivo, adaptado para múltiplas resoluções

## 🎯 Objetivo
O objetivo deste projeto é desenvolver um menu suspenso (dropdown) utilizando técnicas avançadas de CSS, com ênfase em:

- **Animações:** Implementar transições suaves e efeitos visuais que melhoram a experiência do usuário ao interagir com o menu.
- **Posicionamento Absoluto:** Aplicar a propriedade position: absolute para controlar o posicionamento dos elementos do menu, garantindo que ele se comporte de maneira responsiva e se adapte a diferentes layouts.
Ao concluir este projeto, espera-se adquirir um entendimento mais profundo sobre como criar menus interativos e esteticamente agradáveis, além de aprimorar as habilidades em CSS.

## 📋 Requisitos
🔹 **Variáveis CSS** 🔹 **Position absolute** 🔹 **Manipulação DOM**

## ✨ Funcionalidades
- Alterna o ícone de menu para "fechar" ao abrir a barra lateral
- Exibe submenus ao passar o mouse sobre categorias
- Transições suaves e animações interativas

## 📝 Como funciona
1. Navegue pelos submenus para acessar cursos e tutoriais específicos.
2. No mobile, clique no ícone de menu para expandir ou recolher a barra lateral.
3. A barra lateral se adapta automaticamente ao tamanho da tela, garantindo uma experiência consistente.
